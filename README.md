Static website deployed on NGINX web server running on Ubuntu Linux VM
# NGINX Linux Server

## Descripción
Proyecto realizado en Ubuntu mediante una máquina virtual utilizando VMware Workstation.

Se instaló y configuró NGINX como servidor web para desplegar una página HTML personalizada.

## Herramientas utilizadas
- Ubuntu
- VMware Workstation
- NGINX
- HTML/CSS

## Actividades realizadas
- Instalación de NGINX
- Configuración del servidor web
- Creación de una página web básica
- Verificación del servicio mediante localhost

## Resultado
Se logró desplegar una página web funcional en un servidor Linux utilizando NGINX.
